1.打开一个操作终端

 alt+ctrl+t

用户名@ 主机名 ：当前的操作路径 $ 命令行

2.基本shell指令

​ 

1.ls 显示当前目录下的文件 
  ls -l 显示文件属性信息
 -rwxr-xr-x 1 root root 1037528 May 16  2017 bash
 - 普通文件  文本 多媒体 。。。。
 c 字符设备文件 
 b 块设备文件  存储设备
 d 目录文件 
 l 链接文件 快捷方式  
 p 管道文件 进程通信
 s 网络文件 网络通信
 
2.cd 切换路径（打开文件夹）
  cd 目录名
  
  ~   家目录快捷方式  /home/gec 等价与 ~  
  /   根目录  
  ..  返回上一级目录   
  
3.touch  新建普通文件
      touch 文件名  

4.mkdir  新建目录

   mkdir 目录名      

5.cp 复制文件
 
  cp 目标文件 目标路径
       gec@ubuntu:~$ cp 1.txt xa
  
  **复制文件夹？？
   指令后面加上 -rf
  gec@ubuntu:~$ cp 123 xa

cp: omitting directory ‘123’
gec@ubuntu:~$ cp 123 xa -rf

6.mv 移动文件 、重命名
  
  移动文件
   mv 目标文件名 目标路径
   
   gec@ubuntu:~/xa$ mv 1.txt 123

  ***移动文件夹 ？？？不需要 -rf  
   
   重命名
       mv 旧名字 新名字
       
       gec@ubuntu:~/xa/123$ mv 1.txt 2.txt
       gec@ubuntu:~/xa/123$ ls
       2.txt

7.rm 删除文件
  
  rm 文件名

​ 

 **删除目录文件 ？？？ 
      
   gec@ubuntu:~/xa$ rm 123 -rf

共享文件夹

windows ：编写代码
Linux ：  编译程序

通过vm 新建共享目录

1.先在windows上新建一个文件夹 share （记住位置）

2.通过vm 把 share 设置成共享目录

点击 palyer —》管理 —》虚拟机设置 —-》选择 选项 —-》
点击 共享文件夹 —》右侧 选择 总是启用—-》删除掉原来的 目录节点—》
添加 刚才创建的 share

3.访问共享文件夹
windows
linux：
cd /mnt/hgfs
ls 查看是否有刚才新建的share目录

linux应用开发步骤

1.编写 c 代码 （windows）

2.把写好的c程序保存到 共享文件夹

3.进入Linux系统 把c代码编译成 可执行程序

 gec@ubuntu:/mnt/hgfs/share$ gcc demo.c -o demo  

 gcc demo.c -o demo

4.运行程序
./demo
